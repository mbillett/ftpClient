# FTP Client
CS 410/510 Agile FTP Client 

## Building and Running

Git clone the project into Visual Studio IDE and <br>
run the code from there. <br><br>

Or from the command line, compile the program.cs file:

```
$ csc program.cs
```

and then run it:

```
$ mono program.exe
```
